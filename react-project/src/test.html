How to Use the App

Hotkey Listen

The Hotkey Listen feature allows you to quickly set a hotkey for capturing moments during your streams. When you click the "Hotkey Listen" button, the app will listen for a key press and assign that key as the hotkey.

Steps:

  1. Click on the "Hotkey Listen" button.
  2. Press the key you want to assign as the hotkey.
  3. The app will confirm the selected key.

Clips Created

All the clips you create during your stream are displayed in the Clips Created section. You can see details like the clip title, date, and game category. Clicking on a clip lets you perform actions such as viewing, editing, or deleting it.

Features:

  • View the list of created clips.
  • Select a clip to see more details or take action.
  • Manage your clips easily with intuitive controls.

View Clip

The View Clip section provides details about a specific clip. Here, you can edit the clip's title, see a thumbnail, or remove it from the list. Use the "Next" button to navigate to the next clip in the list.

Steps to Edit a Clip:

  1. Select a clip from the "Clips Created" section.
  2. Edit the title or other details.
  3. Save changes or delete the clip if no longer needed.
